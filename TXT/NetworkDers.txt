ETHERNET DATA LINK LAYER
Media Access Control MAC Address
48 bit (6 bytes or 12 digit hex) hardware number 
00-00-00-00-00-00
First 3 byte for the manufacturer
Last 3 byte for the user

Unicast Mac         One sender, one receiver.
Multicast Mac       One sender, multi receiver.
Broadcast           One sender, all of the receiver of the network.
Broadcast address is FF:FF:FF:FF:FF:FF
Cast can not go through router. Only hubs and switches.
